# SpaceX Falcon 9 Launch Success Prediction

## Project Overview

This project analyzes historical SpaceX Falcon 9 launch data to understand the factors that influence first-stage landing success.

The analysis integrates data from the SpaceX API and web-scraped launch records, followed by data cleaning, exploratory data analysis, geospatial mapping, clustering, machine learning modeling, and dashboard visualization.

The goal is to explore patterns behind launch success and build predictive models that estimate whether a Falcon 9 booster will land successfully.

---

## Data Sources

- SpaceX Public API
- Wikipedia Falcon 9 Launch Records (web scraped)

---

## Project Workflow

1. Data Collection from SpaceX API
2. Web Scraping Falcon 9 Launch Records
3. Data Cleaning and Preprocessing
4. Feature Engineering
5. Exploratory Data Analysis
6. Geospatial Mapping using Folium
7. Clustering of Launch Sites
8. Machine Learning Modeling
9. Model Evaluation
10. Interactive Dashboard Development
